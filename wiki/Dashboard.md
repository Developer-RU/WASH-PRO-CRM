# Dashboard

Полное описание: [docs/dashboard.md](https://github.com/Developer-RU/WASH-PRO-CRM/blob/main/docs/dashboard.md) в репозитории.

## Меню

| Группа | Разделы |
|--------|---------|
| Главное | Обзор |
| Объекты | Автомойки, Посты, Текущее состояние |
| Карты | Скидочные, Сервисные, VIP |
| Аналитика | Статистика использования, Финансовая статистика, Архивирование |
| Система | Уведомления, Резервные копии, Telegram, Валюты, Типы скидок, Логи |

Внизу сайдбара — **Resources**: документация и репозиторий.

## Live-обновление

| Интервал | Страницы |
|----------|----------|
| 3 с | Текущее состояние |
| 5 с | Обзор |
| 10 с | Посты, карты, аналитика |
| 15 с | Справочники, архив, бэкапы |

## Ключевые особенности

- **Обзор:** KPI и графики; финансы/использование — сумма последних записей по каждому посту
- **Состояние постов:** баланс, сумма скидки, live-таймер режима
- **Карты:** тип скидки отображается по справочнику 1–5
- **Таблицы:** поиск, сортировка, фильтры, пагинация, экспорт CSV
- **Валюты / типы скидок:** только Administrator
